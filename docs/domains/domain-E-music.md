## E.1 — Schenkerian Analysis: Hierarchical Prolongation

### SOURCE
Schenker (1935), *Der freie Satz* (Free Composition), translated by Oster (1979).
Lerdahl & Jackendoff (1983), *A Generative Theory of Tonal Music*.
Marsden (2010), "Schenkerian Analysis by Computer," *JNMR*.

### FORMAL STRUCTURE EXTRACTION

**Old format:** Music as a flat sequence of notes (pitch, duration, time). Each note is independent. No hierarchical structure — a melody is just an ordered list of events.

**New format:** Music as a hierarchical tree of prolongations from a deep structure (Ursatz). The Ursatz is a minimal skeleton (I-V-I with descending line). Surface music is generated by recursive prolongation: neighbor notes, passing tones, arpeggiations, linear progressions. Each level adds detail while preserving structural tones of the level above.

**Transition op:** Recursive top-down elaboration from Ursatz. Select a structural interval, apply a prolongation rule (e.g., passing tones), each inserted tone becomes eligible for further prolongation. Key property: **prolongation preserves structural tones** — removing deeper levels recovers the structure above.

**Became expressible:** Multi-level structural analysis. Long-range tonal dependencies. Structural note significance (depth in prolongation tree). Compositional coherence from a single generating structure.

**Computable:** YES — Rule application O(1) per step. Reverse analysis partially automatable (Marsden 2010).

### CAGE DIAGNOSIS
**Layer affected:** Layer 2 (Grammar) + partial FORMAT_CHANGE
**Within-format:** BORDERLINE — ExprNode trees are hierarchical but lack structural preservation under level-stripping. Prolongation requires that composition rules at each level respect structural tones of the level above.

### TRANSPLANT SKETCH
Add ProlongationRules to GrammarLayer that elaborate between "preserved" structural nodes. Define "structural depth" as MAP-Elites descriptor.

### VERDICT: STRUCTURAL_EXPANSION

## E.2 — Serialism and Twelve-Tone Technique

### SOURCE
Schoenberg (1941/1975), *Style and Idea*.
Babbitt (1961), "Set Structure as a Compositional Determinant," *JMT*.
Morris (1987), *Composition with Pitch-Classes*.

### FORMAL STRUCTURE EXTRACTION

**Old format (Tonal):** Hierarchical pitch system based on tonic. Asymmetric functional roles.

**New format (Twelve-tone):** Tone row generates 48 forms via group D₁₂: {P, I, R, RI} × {T₀...T₁₁}. Completely symmetric pitch organization.

**Transition op:** Choose row, apply group operations to generate 48 forms, compose by selecting forms. All pitch material derives from row.

**Became expressible:** Symmetric pitch organization. Combinatorial row relationships. However: any serial composition is re-expressible as pitch-class sequence — same F_theo as tonal.

**Computable:** YES — Group operations O(1) per pitch.

### CAGE DIAGNOSIS
**Layer affected:** Layer 1+2. **Within-format:** YES — 12 pitch classes unchanged. Group operations are bijections on same set. Re-parameterization, not expansion.

### TRANSPLANT SKETCH
Group operations as PrimitiveOp transformations. Useful for F_eff (search symmetry reduction by factor ~48).

### VERDICT: COMBINATORIAL_RECOMBINATION

## E.3 — Spectral Music: Timbre as Compositional Parameter

### SOURCE
Grisey (1987), "Tempus ex Machina," *CMR*.
Murail (2005), "The Revolution of Complex Sounds," *CMR*.
Fineberg (2000), "Guide to Spectral Music," *CMR*.

### FORMAL STRUCTURE EXTRACTION

**Old format (Note-based):** Discrete events: pitch (12-TET), duration, velocity. Timbre is fixed property of instrument, not compositional parameter.

**New format (Spectral):** Sound as continuous spectrum: S = [(f₁,a₁,φ₁), (f₂,a₂,φ₂), ...]. Composition operates on spectral transformations: morphing, filtering, stretching harmonic series.

**Transition op:** FFT decomposition → spectral vector representation → compose by operating on spectral vectors.

**Became expressible:** Timbre as continuously variable parameter. Gradual timbre transformations. Inharmonic spectra. Microtonal pitch from overtone relationships. Space expands from {f: R → discrete} to {f: R → R^k}.

**Computable:** YES — FFT O(n log n). Spectral manipulation O(k) per partial.

### CAGE DIAGNOSIS
**Layer affected:** Layer 1 + FORMAT_CHANGE
**Within-format:** NO — Requires continuous-valued, vector-valued primitives. Current ExprNode has discrete scalar output only.

### TRANSPLANT SKETCH
Extend PrimitiveOp to output vectors (SpectralOp). Add spectral composition rules (SpectralMix). MAP-Elites descriptors: spectral richness, spectral centroid.

### VERDICT: STRUCTURAL_EXPANSION

## E.4 — Generative Theory of Tonal Music (GTTM)

### SOURCE
Lerdahl & Jackendoff (1983), *A Generative Theory of Tonal Music*.
Hamanaka et al. (2006), "Implementing GTTM," *JNMR*.

### FORMAL STRUCTURE EXTRACTION

**Old format:** Single-dimension analysis (harmonic OR rhythmic). No formal interaction between dimensions.

**New format:** Four interdependent hierarchical components: grouping structure, metrical structure, time-span reduction, prolongational reduction. Each applies well-formedness rules (WFRs, hard) + preference rules (PRs, soft ranking).

**Transition op:** Four components interact in dependency order. Each output constrains next input. Preference rules rank among well-formed alternatives.

**Became expressible:** Multi-dimensional structural analysis. Preference-based disambiguation. Emergent structural importance. Cross-dimensional constraints.

**Computable:** YES for WFRs (O(n²)). PARTIALLY for PRs (NP-hard, practical heuristics exist).

### CAGE DIAGNOSIS
**Layer affected:** Layer 2 — multi-component hierarchy with preference-based selection.
**Within-format:** STRUCTURAL_EXPANSION — Preference rule mechanism is new (replaces random selection). Four parallel trees with cross-constraints inexpressible in single-tree ExprNode.

### TRANSPLANT SKETCH
Replace random rule selection with preference-ranked selection (connects to C.1b paribhāṣā). Multi-component hierarchy: four parallel ExprNode trees per individual with cross-constraints.

### VERDICT: STRUCTURAL_EXPANSION

## E.5 — Algorithmic Composition: L-Systems

### SOURCE
Lindenmayer (1968), "Mathematical Models for Cellular Interaction," *J. Theor. Bio.*
Prusinkiewicz & Lindenmayer (1990), *The Algorithmic Beauty of Plants*.
McCormack (1996), "Grammar-Based Music Composition," *Complex Systems*.

### FORMAL STRUCTURE EXTRACTION

**Old format (Sequential/CFG):** One non-terminal rewritten per step. Order matters.

**New format (L-system):** ALL symbols rewritten simultaneously per step. Axiom ω + rules → exponential growth. Parametric L-systems add numeric parameters.

**Transition op:** Replace sequential with parallel derivation. For each symbol simultaneously, apply matching rule.

**Became expressible:** Self-similar (fractal) structures from simple rules. Exponential growth from minimal specification. Context-sensitive variants generate structures impossible with context-free sequential grammars.

**Computable:** YES — O(|string| × max_rule_length) per step.

### CAGE DIAGNOSIS
**Layer affected:** Layer 3 (Meta-Grammar).
**Within-format:** YES — Parallel rewriting generates same language class as sequential for DOL-systems. Exponentially more compact for self-similar structures (F_eff), not more expressive (F_theo).

### TRANSPLANT SKETCH
L-system as MetaGrammarLayer strategy: expand ALL grammar rules simultaneously (coordinated expansion). Parametric L-systems connect to A.4 (VW two-level grammars).

### VERDICT: COMBINATORIAL_RECOMBINATION

## E.6 — Microtonality and Just Intonation

### SOURCE
Partch (1974), *Genesis of a Music* (2nd ed.).
Erv Wilson, "Combination Product Sets" (Wilson Archives).
Sethares (1999), *Tuning, Timbre, Spectrum, Scale*.

### FORMAL STRUCTURE EXTRACTION

**Old format (12-TET):** 12 equally spaced pitch classes. Cyclic group Z₁₂. All intervals approximate.

**New format (JI lattice):** Pitches as rational ratios. Multi-dimensional lattice: 5-limit = 2D (axes: fifths 3/2, thirds 5/4). 7-limit = 3D. Each new prime adds a dimension. Lattice is infinite.

**Transition op:** Replace 12-TET approximation with exact rational ratios. Choose prime limit, generate lattice, select finite region as scale, compose using lattice distances as consonance measure.

**Became expressible:** Pure intervals (exact 3/2, 5/4, 7/4). Intervals absent from 12-TET (7/4 harmonic seventh, 11/8). Commatic distinctions (81/80, 64/63). Lattice-theoretic harmonic relationships.

**Computable:** YES — Rational arithmetic exact. Lattice generation O(n^d). Consonance O(k²).

### CAGE DIAGNOSIS
**Layer affected:** Layer 1 + partial FORMAT_CHANGE
**Within-format:** STRUCTURAL_EXPANSION — Infinite lattice genuinely expands vocabulary. Lattice geometry provides compositional constraints absent from flat vocabulary.

### TRANSPLANT SKETCH
Simple: add JI ratios as PrimitiveOps with cost = lattice distance. Structured: LatticeVocabulary with consonance-based composition constraints.

### VERDICT: STRUCTURAL_EXPANSION
